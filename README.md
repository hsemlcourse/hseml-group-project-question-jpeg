## Описание задачи

Проект посвящен разработке модели машинного обучения для предсказания стоимости мотоциклов на вторичном рынке Европы на основе их характеристик (пробег, мощность, тип топлива, марка и т.д.).

**Задача:** Регрессия

**Датасет:** Europe Motorbikes (Zenrows)

**Целевая метрика:** 
* **MAE (Mean Absolute Error)** — основная метрика, так как она легко интерпретируется в реальных деньгах (показывает, на сколько евро в среднем ошибается модель).
* **RMSE (Root Mean Squared Error)** — вспомогательная метрика для штрафования модели за сильные отклонения.

## Запуск

Для воспроизведения результатов проекта выполните следующие шаги:

```bash
# 1. Клонировать репозиторий
git clone <url-вашего-репозитория>
cd <имя-папки>

# 2. Создать виртуальное окружение
python -m venv .venv
source .venv/bin/activate   # Linux/macOS
# .venv\Scripts\activate    # Windows

# 3. Установить зависимости
pip install --upgrade pip setuptools wheel
pip install -r requirements.txt

# 4. Подготовить данные (убедитесь, что исходный csv лежит в data/raw/)
python src/make_dataset.py
python src/build_features.py

# 5. Обучить Baseline-модель
python src/train_baseline.py
```
